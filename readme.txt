【機能】
設定したキーを押すことで素早くXMBに戻れます。（HOMEボタンを押して戻るより断然はやいです。）
おまけとしてシャットダウン、再起動、スリープ、ゲームリセットなども出来ます。

-------------------------------------------------------------------------------------
【動作環境】
5.00系〜6.60系

-------------------------------------------------------------------------------------
【導入方法】
「FastExitGame.prx」と「FastExitGame.ini」をms0(ef0):/sepluginsにコピー
「vsh.txt」、「game.txt」、「pops.txt」の好きなものに

ms0(ef0):/seplugins/FastExitGame.prx 1

と追記

-------------------------------------------------------------------------------------
【使い方】
「FastExitGame.ini」を開いて「Bootkey」「Rebootkey」「Shutdownkey」「Sleepkey」「Resetky」の部分を好きなボタンに変えて保存してください。
何も記入しなければ無効となります。
詳しくは「FastExitGame.ini」を見てください

※Bootkeyを空欄にしている場合Bootkeyは「START」+「HOME」となります。

Rebootkey   = XMBに戻る / 再起動（HENの場合はRESTART VSH）
Shutdownkey = シャットダウン（HENの場合はスリープ）
Sleepkey    = スリープ
Resetkey    = ソフトリセット（ゲームをリロードします）

ボタン名は以下の中から設定してください。
何も記入しなければ無効となります。
キーの数は 「+」 で繋げば、いくらでも指定できます

SELECT      | SELECT
START       | START
↑          | UP
→          | RIGHT
↓          | DOWN
←          | LEFT
Lトリガー   | LTRIGGER
Rトリガー   | RTRIGGER
△          | TRIANGLE
○          | CIRCLE
×          | CROSS
□          | SQUARE
HOME        | HOME
♪          | NOTE
スクリーン  | SCREEN
音量+       | VOLUP
音量-       | VOLDOWN

最後は必ず改行して下さい
